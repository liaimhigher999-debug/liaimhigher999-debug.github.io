# Visual assets

The album cover anchors were downloaded from the official The 1975 Bandcamp
release pages and are credited in the interface. All framing, texture,
typography, and era-specific atmosphere are original CSS treatments for this
non-commercial fan tribute.

Before a commercial release, replace remote artwork with explicitly licensed
local assets and update the credit list in `src/data/eras.ts`.

Local audio, photos, and videos belong in `user-media/`. See its README for the
supported slots. User-provided media is intentionally ignored by source control.
